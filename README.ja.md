# mp4 Slicer

ブラウザ上で簡単かつ高品質に、MP4動画からPNGフレームを抽出できる特化型ツールです。すべての処理はブラウザ内（クライアントサイド）で完結するため、動画がサーバーにアップロードされることはありません。

[English Version (README.md)](README.md)

## ✨ 特徴

- **プライバシー保護**: すべての処理をローカルで実行。動画データは外部に送信されません。
- **高機能クロップ**: 
  - 自由指定のほか、アスペクト比プリセット（16:9, 4:3, 1:1, 9:16）に対応。
  - ドラッグ操作で直感的に範囲を指定可能。
- **精密な時間指定**: 
  - 視覚的なレンジハンドルで開始・終了地点を選択。
  - ミリ秒単位の直接入力による詳細指定も可能。
- **柔軟な間隔設定**: フレームの抽出間隔（例：0.5秒ごと、1.0秒ごと）を自由に設定。
- **一括書き出し**: 
  - 書き出し前にプレビューで確認可能。
  - 抽出したフレームをまとめてZIP形式でダウンロード。
- **モダンなUI**: 
  - 洗練されたグラスモーフィズム・デザイン。
  - ダークモードとライトモードの切り替えに対応。
  - レスポンシブレイアウト。

## 🚀 使い方

1. `index.html` をブラウザで開きます。
2. MP4ファイルをアップロードエリアにドラッグ＆ドロップします。
3. 必要に応じてクロップ範囲や時間範囲を調整します。
4. **「プレビュー生成」**をクリックして、抽出されるフレームを確認します。
5. **「PNGをダウンロード (ZIP)」**をクリックして保存します。

## 🛠️ 使用技術

- **HTML5 / CSS3**: パフォーマンスを重視したバニラ実装。
- **JavaScript (ES6+)**: コアロジックおよびCanvasを用いたフレーム抽出。
- **[JSZip](https://stuk.github.io/jszip/)**: クライアントサイドでのZIP生成。
- **[Lucide Icons](https://lucide.dev/)**: クリーンでモダンなアイコンセット。
- **Google Fonts**: 視認性に優れた `Inter` フォントを採用。
